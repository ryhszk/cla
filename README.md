# Description

cla is a Command line based LAuncher.

(The name 'cla' reminded me of a CoeLAcanth. so, the above image)

# Support OS

- Linux
- Windows (TBD)

# Installation

```
$ git clone https://github.com/ryhszk/cll
$ cd cll
$ go install
```
